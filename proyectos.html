Proyecto 1: Web Educativa - COMED CODERS

articipé activamente como Diseñadora Gráfica y desarrolladora Front-end en un proyecto innovador para el NASA Space Apps Challenge, enfocado en la enseñanza del espacio a niños a través de realidad virtual. Mis responsabilidades clave incluyeron la concepción y creación de la identidad visual completa del proyecto, abarcando desde el diseño de logotipos y paletas de colores hasta la ilustración de personajes y entornos virtuales inmersivos. Adicionalmente, fui responsable del diseño y desarrollo front-end integral de la página web del proyecto utilizando HTML y CSS, asegurando una interfaz intuitiva, visualmente atractiva y responsive para comunicar eficazmente la propuesta educativa. Mi rol dual garantizó una coherencia estética y funcional entre los elementos visuales de la realidad virtual y la presencia online del proyecto.
Tecnologías: HTML, CSS...
Herramientas: Blender, Photoshop, Illustrator...

Proyecto 2: Red Social - QuickTalk

Desarrollé una red social fullstack inspirada en Twitter y Reddit, centrada en el intercambio conciso de ideas y la creación de comunidades. Utilicé Django (backend) para implementar la lógica del servidor, incluyendo gestión de usuarios, publicaciones, interacciones (likes, comentarios) y seguidores, optimizando la base de datos. En el frontend, con HTML, CSS y Bootstrap, diseñé una interfaz intuitiva y responsive. Este proyecto demostró mis habilidades integrales en el desarrollo backend y frontend, creando una plataforma funcional y escalable centrada en la experiencia del usuario.
Tecnologías: Python (Django), HTML, CSS, Bootstrap, JS.

Proyecto 3: Página de Películas con API - Nekoflix

Sitio web construido con HTML, CSS y JavaScript para la exploración interactiva de información cinematográfica a través del consumo de una API de películas. La plataforma permite a los usuarios acceder a detalles exhaustivos de cada filme, incluyendo su sinopsis, elenco, calificaciones e imágenes promocionales. Se diseñó e implementó una interfaz de usuario intuitiva y visualmente atractiva, utilizando el framework Bootstrap para asegurar una presentación organizada, adaptable a diferentes dispositivos y una navegación fluida a través del catálogo de películas.
Tecnologías: HTML, CSS, JS, Bootstrap.
Api: OMDb API - The Open Movie Database
